# Superstore Tableau Dashboard

This project presents an interactive Superstore Sales Performance Dashboard built using Tableau.

## Features
- KPI metrics for Sales, Profit, Quantity, and Customers
- Category and Sub-Category sales analysis
- Top 5 Products by Sales
- Top 5 States by Sales
- Interactive filters for better data exploration
- Use of Calculated Fields and LOD Expressions

## Tools Used
- Tableau
- Superstore Dataset

## Project Purpose
The dashboard helps analyze sales trends, identify top-performing products and regions, and support data-driven decision making.
